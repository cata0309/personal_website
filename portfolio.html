Jarvs

2016   UCSF Memory and Aging Center

Python, Bash, SQLite


Jarvs is an electronic personal assistant and RVS management system. This app includes the RVS scripts below and was a necessary step in making them available to those who don't code or use the terminal. Jarvs also holds all the different scripts together and links them to a scalable, dynamic database so that it can run in different studies of different size. Check out the github repository for more information or your own JARVS.

This app is still in pre-alpha and future features will include:

  • Intuitive database management for attendings
  • Full terminal and gui functionality
  • Clickable binary executable
  • Comprehensive tracking of fellows and archive with a table-response for each rvs
